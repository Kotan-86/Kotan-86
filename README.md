## 代表作
### YYChat
[ろう・難聴者向けに聴者と会話する際に、チャット入力感覚で会話できるように支援するシステム](https://github.com/Kotan-86/yychat)
* 聴者と雑談する際、いちいち文字を見せるのも手間で全部打ち終わるのを待たせたくないという問題意識から出発したプロダクト
* コア機能は以下2つ。
  * 漢字変換確定で入力内容が自動で音声で読み上げられる機能
  * フリック入力と削除で、異なる効果音が出る機能
* これによって、以下の価値が生まれることを目指している。
  * ろう・難聴者は、チャット入力感覚で言いたいことが言える。
  * 聴者は、音声会話と近い感覚で聞けるかつ、入力様子を見なくても入力内容の長さや入力が続きそうということが分かる。
  * その結果、お互いの発言手段が違っても、スムーズにコミュニケーションできる状態を作れる。
